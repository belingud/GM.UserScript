# GM.UserScript

[中文](README.md)

This project is used to store commonly used Tampermonkey userscripts.

Current scripts:

- `enhanced-reverse-image-search.js`: an enhanced reverse image search script that supports local files, pasted URLs, and clicking images on pages, with multiple search engines.
- `auto-scroll.js`: an auto random scrolling script. You can customize target websites with `@match`, and it provides a floating button plus a config panel to start/stop scrolling and save settings. (`@match` is defined in the script header.)

Detailed documentation is in `docs/`:

- [以图搜图增强版（中文）](docs/enhanced-reverse-image-search.md)
- [Enhanced Reverse Image Search (English)](docs/enhanced-reverse-image-search_en.md)
