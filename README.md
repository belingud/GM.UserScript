# GM.UserScript

[English](README_en.md)

这个项目用于保存常用的 Tampermonkey（油猴）脚本。

当前包含的脚本：

- `enhanced-reverse-image-search.js`：以图搜图增强版，支持本地文件、图片链接、点击网页图片三种方式发起搜图，并可跳转多个搜图引擎。
- `auto-scroll.js`：自动随机滚动脚本，可使用`@match`自定义目标网站，提供悬浮按钮与配置面板，可开始/停止滚动并保存配置。（`@match`在脚本头部定义）

详细文档位于 `docs/`：

- [以图搜图增强版（中文）](docs/enhanced-reverse-image-search.md)
- [Enhanced Reverse Image Search (English)](docs/enhanced-reverse-image-search_en.md)
- [自动随机滚动（中文）](docs/auto-scroll.md)
